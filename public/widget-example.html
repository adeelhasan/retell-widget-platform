data-target Placement

Use data-target to render the widget inside a specific element instead of after the script tag. 

<div id="my-demo-slot"></div>
<script
  src=".../api/widget-simple"
  data-widget-id="..."
  data-target="my-demo-slot">
</script>
RetellWidget.open() JS API

Trigger a widget call from any custom button or JS event using RetellWidget.open(widgetId). 

<script src=".../api/widget-simple"
  data-widget-id="abc123"></script>

<button onclick="RetellWidget.open('abc123')">
  Custom CTA Button
</button>
